## Download

GitHub clone:

```
$ git clone git@github.com:ecomfe/san.git
```

NPM:

```
$ npm i san
```

CDN:

```html
<!-- unpkg -->
<script src="https://unpkg.com/san@latest"></script>

<!-- compressed -->
```
